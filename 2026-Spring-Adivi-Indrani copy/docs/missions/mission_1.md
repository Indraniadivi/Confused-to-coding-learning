## What is a Variable?

A variable is a **named container** for storing information.
Think of it like a labeled box:
- the **label** is the variable name
- the **thing inside** is the value

In Dart it looks like this:
```dart
String astronautName = "Indrani";
int missionNumber = 1;
double fuelLevel = 98.5;
bool isReady = true;
```

## The 4 Basic Data Types in Dart

| Type   | What it stores | Example.         |
|--------|----------------|------------------|
| String | Text.          | "Planet Caloria" |
| int    | Whole numbers  | 7                |
| double | Decimal numbers| 98.5             |
| bool   | True or False  | true             |

## Why This Matters
Variables help you store information so your app can use it later.
For example:
- a name
- a score
- a number
- a yes/no choice

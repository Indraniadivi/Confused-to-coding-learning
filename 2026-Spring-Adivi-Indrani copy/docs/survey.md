# Pre/Post Survey (Beginner User Testing)

This survey is designed to evaluate:

**Research Question:**  
*Can a mission-based learning platform enable individuals with no prior coding knowledge to successfully build and understand a working mobile application?*

---

## Instructions
- Give the **Pre-Survey** before learners start the platform.
- Give the **Post-Survey** after learners complete the missions.
- Questions 1–5 are repeated to measure learning gain.

---

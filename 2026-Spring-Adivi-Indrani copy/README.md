# Gamified Health & Nutrition Learning Platform

## Project Overview
This repository hosts my Master’s Project, focused on the development of a gamified application designed to improve nutritional literacy. While traditional trackers focus strictly on data entry, this platform utilizes game mechanics—such as leveling systems and interactive challenges—to educate users on macronutrients and caloric balance.

## Project Objective
The objective of this project is to design, develop, and evaluate a system that:
* **Gamifies Healthy Habits:** Converts daily nutritional logging into an interactive experience.
* **Educational Integration:** Provides active feedback on caloric balance and nutrient density.
* **User Retention:** Leverages engagement loops (XP and leveling) to encourage long-term health management.

## Proposed Approach
The application is built using a cross-platform framework, currently optimized for web-based interaction:
* **Frontend:** [Flutter](https://flutter.dev) for a responsive UI.
* **Logic:** [Dart](https://dart.dev) for calorie calculations and progress tracking.
* **Data Integration:** Fetching real-time nutritional data via specialized APIs.

## Technologies
* **Flutter & Dart** (App Development)
* **Python** (Backend data processing)
* **Nutrition APIs** (e.g., Edamam)
